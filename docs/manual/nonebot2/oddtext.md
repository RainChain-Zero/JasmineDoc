# 奇怪文字转换

> _茉莉？...你真的没事吗？...可是都乱码了诶_

也许这要归功于茉莉偶尔奇怪的故障(？你有时能看到奇怪的文字乱码。

### "/抽象话 [文字]"

### "/火星文 [文字]"

### "/蚂蚁文 [文字]"

### "/翻转文字 [文字]"

该指令仅支持英文。
